## 진행방법
1. 저장소 fork 
   1. 자신의 환경에 맞게 클론
   2. 예) 인텔리제이 터미널에서 `git clone {https code}`
2. 브랜치 설정
   1. `git checkout -b 브랜치이름` 예) `git checkout -b fortuneman`
3. 설정된 브랜치에서 진행
4. 1주 1PR 원칙 하에 자율적으로 활동
   1. 스스로 기초를 다듬고 싶은것 아무거나
   2. 4를 지킨다면 무엇을 해도 상관없음
